The D2RML-FN Vocabulary

This document defines the Data to RDF Mapping Language Functions (D2RML-FN) vocabulary.

Namespace

The namespace is http://islab.ntua.gr/ns/d2rml-fn.

Classes

This section defines the classes of this vocabulary.

drfn:Function

A function.

URI:
http://islab.ntua.gr/ns/d2rml-fn#Function

drfn:FunctionDescription

A description of a function.

URI:
http://islab.ntua.gr/ns/d2rml-fn#FunctionDescription

drfn:Parameter

A parameter description for an entity that takes taking parametric inputs.

URI:
http://islab.ntua.gr/ns/d2rml-fn#Parameter

drfn:Result

A result description for an entity that generated an output.

URI:
http://islab.ntua.gr/ns/d2rml-fn#Result

Properties

This section defines the properties of this vocabulary.

drfn:containedType

The type of the elements of a container a parameter or result.

URI:
http://islab.ntua.gr/ns/d2rml-fn#containedType
Domain:
drfn:Parameter, drfn:Result
Range:
An IRI specifying a datatype in the xsd namespace.

drfn:defaultValue

The default value of a parameter.

URI:
http://islab.ntua.gr/ns/d2rml-fn#defaultValue
Domain:
drfn:Parameter
Range:
A literal.

drfn:describes

The function that a function description describes.

URI:
http://islab.ntua.gr/ns/d2rml-fn#describes
Domain:
drfn:FunctionDescription
Range includes:
drfn:Function

drfn:name

The name of a parameter or result.

URI:
http://islab.ntua.gr/ns/d2rml-fn#name
Domain:
drfn:Parameter, drfn:Result
Range:
An xsd:string literal.

drfn:parameter

A parameter of a function.

URI:
http://islab.ntua.gr/ns/d2rml-fn#parameter
Domain:
drfn:FunctionDescription
Range:
drfn:Parameter

drfn:parameters

A list of parameters. The order of a parameter in the list determines its position.

URI:
http://islab.ntua.gr/ns/d2rml-fn#parameters
Domain:
drfn:FunctionDescription
Range:
An rdf:List of drfn:Parameters

drfn:position

The position of a parameter.

URI:
http://islab.ntua.gr/ns/d2rml-fn#name
Domain:
drfn:Parameter
Range:
An xsd:integer literal.

drfn:required

Used to specify if a parameter is required or optional.

URI:
http://islab.ntua.gr/ns/d2rml-fn#required
Domain:
drfn:Parameter
Range:
An xsd:boolean literal.

drfn:result

A results of a function.

URI:
http://islab.ntua.gr/ns/d2rml-fn#parameter
Domain:
drfn:FunctionDescription
Range:
drfn:Parameter

drfn:results

A list of results. The order of a parameter in the list determines its position.

URI:
http://islab.ntua.gr/ns/d2rml-fn#parameters
Domain:
drfn:FunctionDescription
Range:
An rdf:List of drfn:Parameters

drfn:type

The type of a parameter or result.

URI:
http://islab.ntua.gr/ns/d2rml-fn#type
Domain:
drfn:Parameter, drfn:Result
Range:
An IRI specifying a datatype in the xsd namespace or rdf:List.
