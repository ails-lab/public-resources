The D2RML Expression Languages (D2RML‑EL) Vocabulary

This document defines the D2RML Expression Languages (D2RML‑EL) vocabulary.

Namespace

The namespace drel is http://islab.ntua.gr/ns/d2rml-el#.

Classes

drel:ExpressionLanguage

An expression language.

URI:
http://islab.ntua.gr/ns/dr2ml-el#ExpressionLanguage

Individuals

The currently defined expression languages by D2RML-EL are listed in the following table. All are instances of drel:ExpressionLanguage.

Character Encodings
IRI Description
drel:JSONKey
 A JSON key (name)
drel:JSONPath
 A JSONPath expression
drel:RegEx
 A regular expression in default syntax
drel:RegExJava
 A regular expression in Java syntax containing at least one capturing group
drel:RegExPerl
 A regular expression in Perl syntax containing at least one capturing group
drel:RegExPython
 A regular expression in Python syntax containing at least one capturing group
drel:XMLElement
 An XML element name
drel:XPath
 An XPath expression in default version
drel:XPath10
 An XPath 1.0 expression
drel:XPath20
 An XPath 2.0 expression
drel:XPath30
 An XPath 3.0 expression
drel:XPath31
